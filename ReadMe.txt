Made with Clear engine v3

What clear engine supports:
Texture mapping
Multiple storys
And much much more!

Clear engine is based off of an engine called pyray which can be found here:
https://github.com/oscr/PyRay

the engine now works on python 3

The modules used were:
math
time
pygame
os
random
PIL

You may use the textures that were used in this project and they are located in the dist folder just don't claim the texture are yours.

source code can be found here:
https://github.com/Ultimate458/Maze-game-v2-source-code.git
